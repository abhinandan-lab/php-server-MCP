🚀 PHP Light Framework

A modern, lightweight PHP framework for building robust REST APIs with built-in security, validation, and interactive documentation.

✨ Key Features

🛣️ Flexible Router

Array-based route definitions with URL, GET, POST, and JSON parameter support

🔒 Security First

XSS protection, CORS handling, rate limiting, and comprehensive security headers

🎛️ Request Validation

Built-in input sanitization and validation middleware with custom rules

📚 Interactive Docs

Beautiful API documentation with live testing interface and environment management

💾 Database Integration

Enhanced database operations with named parameters and migration system

🐛 Smart Debugging

Context-aware debug functions with environment controls and error logging

⚡ Quick Start

  1. Install: composer install
  2. Configure: Copy .env.example to .env
  3. Setup Database: Configure database credentials in .env
  4. Run Migration: POST to /api/run_migration with your schema
  5. Start Building: Create routes and controllers

📋 Requirements

  • PHP 8.0 or higher
  • Composer for dependency management
  • MySQL/PDO support
  • Apache/Nginx with URL rewriting
