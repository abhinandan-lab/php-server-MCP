💾 Database

Learn how to work with databases securely using prepared statements, the RunQuery helper, and database migrations.

⚡ Quick Start

// Simple database query
$users = RunQuery([
    'conn' => $this->conn,
    'query' => 'SELECT * FROM users WHERE status = :status',
    'params' => [':status' => 'active']
]);

foreach ($users as $user) {
    echo $user['name'];
}

🔌 Database Connection

Configuration

Database settings are configured in your .env file:

DB_HOST=db
DB_PORT=3306
DB_NAME=testdb
DB_USER=root
DB_PASSWORD=abcd

Accessing Connection

The database connection is automatically available in controllers:

class UserController extends BaseController
{
    public function getUsers()
    {
        // Connection available as $this->conn
        $users = RunQuery([
            'conn' => $this->conn,
            'query' => 'SELECT * FROM users'
        ]);
        
        $this->sendSuccess('Users retrieved', $users);
    }
}

🔍 RunQuery Function

Basic Usage

$result = RunQuery([
    'conn' => $this->conn,           // PDO connection
    'query' => 'SELECT * FROM users', // SQL query
    'params' => [],                   // Parameters (optional)
    'fetchAssoc' => true,            // Return format (optional)
    'withSuccess' => false            // Get metadata (optional)
]);

SELECT Query

$users = RunQuery([
    'conn' => $this->conn,
    'query' => 'SELECT id, name, email FROM users WHERE role = :role',
    'params' => [':role' => 'admin']
]);

// Returns array of rows
// [
//   ['id' => 1, 'name' => 'John', 'email' => 'john@example.com'],
//   ['id' => 2, 'name' => 'Jane', 'email' => 'jane@example.com']
// ]

INSERT Query

$result = RunQuery([
    'conn' => $this->conn,
    'query' => 'INSERT INTO users (name, email, password) 
                VALUES (:name, :email, :password)',
    'params' => [
        ':name' => 'John Doe',
        ':email' => 'john@example.com',
        ':password' => password_hash('secret', PASSWORD_DEFAULT)
    ],
    'withSuccess' => true
]);

// Returns metadata
// [
//   'success' => true,
//   'affected_rows' => 1,
//   'id' => 123  // Last insert ID
// ]

$newUserId = $result['id'];

UPDATE Query

$result = RunQuery([
    'conn' => $this->conn,
    'query' => 'UPDATE users SET name = :name WHERE id = :id',
    'params' => [
        ':name' => 'Updated Name',
        ':id' => 5
    ],
    'withSuccess' => true
]);

if ($result['affected_rows'] > 0) {
    echo "User updated";
} else {
    echo "No changes or user not found";
}

DELETE Query

$result = RunQuery([
    'conn' => $this->conn,
    'query' => 'DELETE FROM users WHERE id = :id',
    'params' => [':id' => 5],
    'withSuccess' => true
]);

if ($result['affected_rows'] > 0) {
    echo "User deleted";
} else {
    echo "User not found";
}

🔒 Security Best Practices

✅ Always Use Prepared Statements

❌ Vulnerable to SQL Injection

// NEVER DO THIS!
$id = $_GET['id'];
$query = "SELECT * FROM users WHERE id = $id";
$result = RunQuery([
    'conn' => $this->conn,
    'query' => $query
]);

✅ Safe with Parameters

// ALWAYS DO THIS
$id = $_GET['id'];
$result = RunQuery([
    'conn' => $this->conn,
    'query' => 'SELECT * FROM users WHERE id = :id',
    'params' => [':id' => $id]
]);
⚠️ Critical: Never concatenate user input into SQL queries. Always use named parameters with RunQuery.

📊 Common Patterns

Check if Record Exists

$existing = RunQuery([
    'conn' => $this->conn,
    'query' => 'SELECT id FROM users WHERE email = :email',
    'params' => [':email' => $email]
]);

if (!empty($existing)) {
    // Email already exists
    $this->sendError('Email already registered', 409);
}

Count Records

$result = RunQuery([
    'conn' => $this->conn,
    'query' => 'SELECT COUNT(*) as total FROM users WHERE status = :status',
    'params' => [':status' => 'active']
]);

$totalUsers = $result[0]['total'];

Search with LIKE

$search = $_GET['search'] ?? '';

$users = RunQuery([
    'conn' => $this->conn,
    'query' => 'SELECT * FROM users WHERE name LIKE :search',
    'params' => [':search' => "%$search%"]
]);

Pagination

$page = $_GET['page'] ?? 1;
$perPage = 20;
$offset = ($page - 1) * $perPage;

$users = RunQuery([
    'conn' => $this->conn,
    'query' => 'SELECT * FROM users 
                ORDER BY created_at DESC 
                LIMIT :limit OFFSET :offset',
    'params' => [
        ':limit' => $perPage,
        ':offset' => $offset
    ]
]);

💡 Tips & Best Practices

Use Named Parameters

Always use :paramName syntax for clarity and safety

SELECT Specific Columns

Avoid SELECT * - specify only needed columns

Use withSuccess

Set withSuccess: true for INSERT/UPDATE/DELETE to get metadata

Handle Empty Results

Always check if query returned results before accessing data
