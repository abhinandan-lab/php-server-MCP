üöÄ Getting Started

Get your PHP Light Framework up and running in minutes with Docker. This guide walks you through the complete setup process from installation to your first API endpoint.

‚ö° Quick Start (60 Seconds)

Already have Docker installed? Run these commands:

git clone https://github.com/yourrepo/php-light-framework.git
cd php-light-framework
docker-compose up -d

# Visit: http://localhost:8080/api/docs
‚úÖ Done! Your framework is now running. Visit http://localhost:8080/api/docs to see the interactive documentation.

üì¶ Installation Steps

1

Clone Repository

Get the framework code from GitHub

git clone https://github.com/yourrepo/php-light-framework.git
cd php-light-framework
2

Configure Environment

Create and edit your .env file

cp .env.example .env
nano .env
3

Start Docker Containers

Launch all services

docker-compose up -d
4

Verify Installation

Check if everything is running

  • API: http://localhost:8080/api/health
  • Docs: http://localhost:8080/api/docs

üéØ Your First API Endpoint

Create Controller

// api/controllers/HelloController.php
<?php
namespace App\Controllers;

class HelloController extends BaseController
{
    public function sayHello()
    {
        $name = $_GET['name'] ?? 'World';
        
        $this->sendSuccess("Hello, $name!", [
            'timestamp' => date('Y-m-d H:i:s'),
            'message' => 'Welcome to PHP Light Framework'
        ]);
    }
}

Register Route

// api/index.php
$router->add([
    'method' => 'GET',
    'url' => '/hello',
    'controller' => [App\Controllers\HelloController::class, 'sayHello'],
    'desc' => 'Simple hello world endpoint'
]);

Test It

Visit: http://localhost:8080/api/hello?name=Developer

üéâ Congratulations! You've created your first API endpoint.
